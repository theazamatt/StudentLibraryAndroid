## Student Library (Android)

Тема: **библиотека для студентов** — каталог книг, выдача/возврат, список выдач.

### Как открыть
- Откройте папку `StudentLibraryAndroid` в Android Studio.
- Если Android Studio попросит создать/восстановить Gradle Wrapper (файл `gradle-wrapper.jar`), согласитесь — это нормальная автоматическая операция.
- Нажмите Run.

### Функции
- Каталог книг с поиском
- Карточка книги
- Выдача книги студенту и возврат
- Экран “Выдачи” (активные/история)
